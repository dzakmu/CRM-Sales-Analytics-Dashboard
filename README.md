# CRM-Sales-Analytics-Dashboard

## 1. Project Overview 
This project presents an interactive CRM Sales Analytics Dashboard developed using Power BI. The dashboard analyzes sales opportunities, revenue performance, sales pipeline, product performance, and sales team effectiveness to support business decision-making through interactive visualizations and KPI monitoring.
## 2. Business Objective
a. Analyze overall sales performance.
b. Monitor sales pipeline conversion.
c. Identify top-performing products and sales agents.
d. Evaluate regional and sector performance.
e. Track revenue trends over time.
f. Measure sales effectiveness using Win Rate.
